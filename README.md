# Shitter

Alternative Twitter App or Android

Features:
fast access to tweets and users
customizable colors
hashtag & username highlighting (also clickable)
image upload  (up to 4 images)
image preview (tweets and profile images)
twitter search

used libraries:
Twitter4j 4.0.6
picasso 2.5.2
flask colorpicker
