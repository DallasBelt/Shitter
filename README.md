# Shitter

Alternative Twitter App for Android

Features:
- fast access to tweets and userprofiles
- customizable colors
- hashtag & username highlighting/clickable
- image upload  (up to 4 images)
- image preview
- twitter trends
- twitter search
- Directmessage
